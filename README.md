# Распознаватель рукописных символов

CNN модель для распознавания рукописных математических символов (цифры, буквы, операторы).

## Возможности

- Распознавание 91 символа (цифры, буквы, математические операторы)
- GUI для рисования символов мышью
- Вывод уверенности предсказания
